# Amazon Sales Analysis Using SQL

## Project Overview
This is a self-driven SQL project that analyzes Amazon sales data from three branches:
Yangon, Mandalay, and Naypyitaw.

The goal of this project is to understand product performance, sales trends,
and customer behavior using SQL.

## Dataset
- 1000 sales records
- 17 columns
- Time period: January to March 2019

## Tools Used
- MySQL Workbench
- SQL
- Microsoft Excel

## Key Analysis Performed
- Product line revenue and VAT analysis
- Monthly and time-of-day sales trends
- Customer type and gender-based analysis
- Branch-wise performance comparison

## Files in This Repository
- `amazon_sales.sql` → All SQL queries used in analysis
- `Amazon_Sales_Report.pdf` → Final project report
- `dataset/` → Sales dataset (CSV)
- `screenshots/` → Query output screenshots

## Key Insights
- Food and Beverages is the top revenue-generating product line
- January recorded the highest sales and COGS
- Member customers contribute the highest revenue
- Sales peak during the afternoon
- Naypyitaw is the highest revenue-generating city
